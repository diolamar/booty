Build EXE (PyInstaller)

Install:

pip install pyinstaller

From script folder:

pyinstaller --onefile register_cli.py

Optional data folder (if needed):

pyinstaller --onefile --add-data "registrations.csv;." register_cli.py

---------------------Build EXE with more Files---------------------------
pyinstaller --onefile \
  --add-data "helpers.py;." \
  --add-data "templates;templates" \
  --add-data "data;data" \
  --add-data "static;static" \
  app.py

or

Non-Python files (images, configs, etc.)
Example:
-> .json
-> .txt
-> .png
  These are NOT included automatically.
  Fix:
  pyinstaller --onefile main.py --add-data "config.json;."
